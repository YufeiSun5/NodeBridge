# V0.8 Persistence

V0.8 建立 ACK、dispatch、error 的持久化基础，并提供失败事件 CLI 入口。

## CLI

```powershell
sync-agent failed-events -config configs/server.example.yaml -limit 50
```

```powershell
sync-agent retry-event `
  -config configs/server.example.yaml `
  -event-id evt-001 `
  -target-node-id edge-002
```

## 完成范围

- `syncstore.Store` 写入 `sync_ack_log`。
- `syncstore.Store` 写入 `sync_dispatch_log`。
- `syncstore.Store` 写入 `sync_error_log`。
- 查询失败事件。
- 将失败事件标记为 `PENDING`，为后续真实重放做准备。
