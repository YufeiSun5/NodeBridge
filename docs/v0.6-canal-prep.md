# V0.6 Canal Prep

V0.6 仍不绑定真实 Canal client。目标是把 Canal 接入前置层稳定下来：

```text
Canal row change -> cdc.ChangeEvent -> offset save -> ack
```

## Smoke

```powershell
sync-agent canal-check -config configs/edge.example.yaml
```

## 已完成范围

- `cdc.Offset` 和 `OffsetStore`。
- `cdc.MemoryOffsetStore`，用于单元测试和开发阶段。
- `internal/cdc/canal` adapter 前置接口。
- Canal row change 到 `cdc.ChangeEvent` 的转换。
- `FetchOnce`：fetch -> convert -> event channel -> save offset -> ack。
- 配置新增 `cdc.reader_name` 和 `cdc.use_gtid`。

## 未完成范围

- 真实 Canal client 依赖选型。
- Canal 网络连接和订阅实现。
- MySQL 持久化 offset store。
- binlog 异常恢复和重连策略。
