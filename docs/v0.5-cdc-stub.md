# V0.5 CDC Stub Smoke

V0.5 先不连接真实 Canal。目标是验证：

```text
ChangeEvent -> Loop Suppressor -> SyncEvent -> Edge local RabbitMQ
```

## Smoke

```powershell
sync-agent publish-change-once `
  -config configs/edge.example.yaml `
  -rules configs/sync-rules.example.yaml `
  -file sample-events/device_config.change.json
```

可显式覆盖本地 RabbitMQ：

```powershell
sync-agent publish-change-once `
  -config configs/edge.example.yaml `
  -rules configs/sync-rules.example.yaml `
  -amqp-url $env:NODEBRIDGE_EDGE_RABBITMQ_URL `
  -file sample-events/device_config.change.json
```

## 范围

- `internal/normalizer` 将 CDC `ChangeEvent` 转成标准 `SyncEvent`。
- `CDCUploadRuntime` 对接 loop suppressor 和 publisher。
- `cdc.StubSource` 用于测试和本地 smoke。
- Canal client、offset 持久化和 binlog 重连不在本版。
