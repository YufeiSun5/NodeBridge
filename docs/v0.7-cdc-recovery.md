# V0.7 CDC Recovery

V0.7 先完成 CDC 可靠性基础，不直接接真实 Canal 网络客户端。

## 完成范围

- MySQL 持久化 offset store：`sync_upload_offset`。
- CDC recovery policy：指数退避、最大延迟、最大次数。
- fatal error 标记：`cdc.ErrFatalRecovery`。
- offset store 单元测试覆盖 load、missing、save、invalid DB。

## 恢复策略

```text
temporary error -> retry with backoff
fatal error -> stop worker and surface status
offset exists -> resume from stored position
offset missing -> start from Canal configured position
```
