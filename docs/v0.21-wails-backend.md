# V0.21 Wails Backend And Tray Support

V0.21 后端目标是支撑 Wails 托盘常驻试用版，不做 Windows Service。

## 已实现

- `SaveConfig` 落盘到 `%ProgramData%\NodeBridge\config.yaml`，可用 `NODEBRIDGE_CONFIG_PATH` 覆盖。
- MySQL/RabbitMQ/CDC/LogWeb 密码、token 和退出密码保存前加密，返回前脱敏。
- `SaveSyncRules` 校验 identifier、主键映射、列映射并写入 `sync-rules.yaml`。
- `GetQueueStatus` 使用 RabbitMQ passive declare 查询队列积压。
- `GetFailedEvents` 和 `RetryFailedEvent` 接入 `sync_ack_log`。
- `GetLogs` 从 runtime ring buffer 读取日志。
- `VerifyExitPassword`、`GetAutoStart`、`SetAutoStart` 为前端托盘退出和登录自启动提供后端能力。
- `ExportDiagnosticPackage` 导出脱敏诊断 zip。
- `scripts/lab-stress-e2e.ps1` 提供 1,000 条默认压力测试入口。

## 边界

- 前端负责托盘菜单、关闭窗口最小化、退出密码弹窗和退出确认。
- 后端不直接实现 Windows 托盘 UI。
- V0.21 不做 Windows Service；后续如需无人登录运行，再单独评估服务化版本。
- Wails UI 默认不占端口；日志 Web 仍是 SyncAgent 独立可选诊断端口。

## 验证

```powershell
go test ./...
go vet ./...
scripts/lab-stress-e2e.ps1 -SkipPrepare
```

`lab-stress-e2e.ps1` 默认使用三套 RabbitMQ 和三套 MySQL lab 环境，发布并处理 1,000 条 `device_config` 事件，验证 Server 和 Edge B 数量、apply log 和失败 ACK。
